# TrackHour

Outil de suivi du temps basé sur Node.js + Express.

## Prérequis

- [Node.js](https://nodejs.org/) (version 18 ou supérieure)

## Installation sur un nouveau PC

1. Cloner ou copier le dossier TrackHour sur le PC
2. Ouvrir un terminal dans le dossier et installer les dépendances :
   ```
   npm install
   ```
3. Double-cliquer sur `TrackHour.vbs` pour lancer l'application

> Le fichier `config.json` sera créé automatiquement lors du premier lancement.
> Il faudra configurer le chemin du dossier de données via l'interface.

## Lancement

- **Démarrer** : double-cliquer sur `TrackHour.vbs`
- **Arrêter** : double-cliquer sur `ARRETER_TRACKHOUR.bat`

L'interface s'ouvre automatiquement sur [http://localhost:3000](http://localhost:3000)
